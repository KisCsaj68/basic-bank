# basic-bank
Basic bank with DI (Google Guice)
Using factory interfaces and factory providers to hide the business logic.

Account types:
- Credit
- Debit

Customer types:
- Person
- Company

Transaction types:
- Deposit
- Withdraw

Basic banking functions:
- Register customer
- Create account(Linked to a customer)
- Cash deposit
- Cach withdraw
- Cach transfer
